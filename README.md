# Bill Splitter

A simple and responsive bill splitter app built with HTML, CSS, and JavaScript. It helps calculate the tip amount, total bill, and each person's share in a few clicks.

## Features

- Enter the bill amount
- Select a preset tip percentage
- Add a custom tip percentage
- Enter the number of people
- Calculate total tip, final bill, and per-person amount
- Reset all values easily
- Responsive layout for mobile and desktop screens

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Enter the bill amount.
2. Select a tip percentage or enter a custom tip.
3. Enter the number of people.
4. Click **Generate Bill** to see the result.
5. Use **Reset** to clear everything.
